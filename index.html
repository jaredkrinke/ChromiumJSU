Chromium JSU is a fast-paced, futuristic shoot 'em up that runs in your browser. It is a rewrite of Chromium BSU (a multi-platform shoot 'em up game created by Mark B. Allan, distributed under the terms of the Clarified Artistic License).

Chromium JSU uses images from the original Chromium BSU release but the code has been rewritten in JavaScript for use in HTML5-capable browsers. Chromium JSU is distributed under the terms of the GNU General Public License.

You can view the source code at the Chromium JSU GitHub repository.
